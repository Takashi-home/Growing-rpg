# Growing-rpg
成長型のRPG
こんな機能が全部入っているゲームを作成して
📖 AI生成ストーリー
毎ターン、AIが現在のHP・レベル・場所・天気・経過日数を把握して唯一無二の物語を紡ぎます。選択肢も毎回変化。
⚔ バトルシステム
・攻撃 / 魔法（MP消費）/ ポーション / 逃走
・勝利でXP＆ゴールドを獲得
・敵の強さはレベルに合わせて変化
🌱 キャラクター進化
・Lv3→Adventurer、Lv5→Knight…最大Legendまで
・レベルアップでHP/MP/ステータス上昇
・スキルや装備をAIがランダム付与
🌍 世界の変化
・場所・天気・日付がリアルタイムで変化
・訪問済みの場所を記憶して連続した世界を構築
選択次第で全く違う冒険になります。楽しんでください！​​​​​​​​​​​​​​​​
なお、共有されたコードでは callAI 関数で Anthropic API を直接呼んでいますが、x-api-key ヘッダーが含まれていないため、そのままでは動作しません。APIキーの入力機能を追加して動くようにして
